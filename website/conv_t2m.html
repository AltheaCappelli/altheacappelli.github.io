Predicting with a convolutional LSTM network the 2m air temperature anomalies in Europe from the Copernicus Climate Data Store (CDS)

The goal of this project is to see if a convolutional LSTM network can be used to predict the 2m air temperature anomalies in Europe. In this case, the temperature grid from Europe is fed to the neural network, which uses a convolutional filter of the size 3x3 to predict the outcome. The model is trained on temperature data between 1940 and 2010, and is tested from 2010 to 2024. Different layer setup and parameters were tested to improve the model's performance. After the model is trained, it is used to predict future temperature anomalies between 2024 and 2035.

I used the 2m air temperature data from the ERA5 Reanalysis dataset provided by the Copernicus Climate Data Store (CDS). ERA5 is a global climate reanalysis dataset that provides hourly estimates of atmospheric variables, including the 2m air temperature.
